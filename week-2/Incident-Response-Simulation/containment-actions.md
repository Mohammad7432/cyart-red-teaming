# Containment Actions

## Objective
To prevent the spread of potential malicious activity.

## Actions Taken

### 1. Process Monitoring
- Identified suspicious PowerShell process

### 2. Isolation
- Recommend isolating the affected system from the network

### 3. Account Review
- Check for unauthorized user activity

### 4. Script Analysis
- Review the executed script for malicious content

### 5. Logging
- Preserve logs for further investigation

## Recommendation
Immediate containment should be followed by deeper analysis to determine if the activity is malicious.
