# Traffic Simulation

## Objective
Simulate network traffic patterns that would trigger Suricata rules.

## Simulated Scenarios

### 1. PowerShell Download
Example:
http://example.com/powershell script

### 2. Command Injection
Example:
cmd.exe /c dir

### 3. Encoded Command
Example:
powershell -enc SQBFAFgA

## Explanation
These simulated patterns represent typical attacker behavior that would be detected by IDS rules.
