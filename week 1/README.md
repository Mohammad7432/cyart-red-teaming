# Week 1 - Red Teaming Lab

## Overview
This project demonstrates a complete Red Teaming assessment performed on a Metasploitable2 virtual machine.

## Objectives
- Perform reconnaissance and scanning
- Identify vulnerabilities
- Exploit system weaknesses
- Demonstrate post-exploitation techniques

## Tools Used
- Nmap
- Metasploit
- Hydra
- Netcat
- KeePassXC
- OpenVAS
- VirusTotal
- Hybrid Analysis

## Target
- Metasploitable2 VM
- IP: 192.168.1.6

## Workflow
The assessment followed a structured Red Team methodology:
Reconnaissance → Scanning → Enumeration → Exploitation → Privilege Escalation → Persistence → Credential Attack → Reverse Shell → Vulnerability Assessment

## Contents
- Report → Full assessment report
- Screenshots → Evidence of tasks
- Notes → Commands used
- Workflow → Step-by-step process

## Summary
This lab demonstrates how multiple vulnerabilities can be exploited to achieve full system compromise and highlights the importance of proper security practices.
